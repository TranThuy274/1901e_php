Sự khác biệt của include, include_once, require, require_once

    Sự giống nhau: $ hàm này đều dùng để nạp 1 file vào 1 file khác
    Sự khác nhau:
    - include: khi nạp file bị lỗi thì chương trình vẫn chạy tiếp
    - require: Khi nạp file bị lỗi thì chương trình sẽ ngừng chạy
    - include_once: giống include nhưng chỉ nạp 1 file 1ần ngay cả khi bị gọi nhiều lần
    - require_one: giống require nhưng chỉ nạp 1 file 1ần ngay cả khi bị gọi nhiều lần
